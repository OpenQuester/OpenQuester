## Guidelines

- Use PNG format for best quality
- Capture screenshots in a clean state (no debug info)
- Use consistent theme (light or dark)
- Avoid personal information in screenshots
- Show the app at its best - clean UI, good content
- Consider adding captions or annotations if helpful

## Usage

Once added, uncomment the image references in the README.md file:

```markdown
![Main Menu](assets/screenshots/main-menu.png)
```
