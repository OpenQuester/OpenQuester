# Welcome to OpenQuester Documentation

OpenQuester is a free, open-source multiplayer quiz game inspired by SIGame. This documentation provides comprehensive information about the game, its features, and technical implementation.

## About OpenQuester

OpenQuester is a multiplayer quiz game that supports:
- Multiple game roles (Showman, Player, Spectator)
- Various question types (Regular, Stake, Secret, No Risk)
- Community-created content packages  
- Cross-platform support (Windows, Linux, Android, and more coming soon)
- Real-time synchronization using WebSocket

## Getting Started

### For Players
1. Download the game for your platform
2. Create an account or play as a guest
3. Join or create a game room
4. Have fun competing with friends!

Check out our [GitHub repository](https://github.com/OpenQuester/OpenQuester) to contribute to the project or report issues.
